project.js       Project Passport logic
moodboard.js     Moodboard drag, resize and pin logic

HOW TO RUN
----------
1. Keep every file in the same folder.
2. Open index.html in Chrome.
3. Click "Create your membership".
4. Log in with the account you created.
5. The login page will open projects.html automatically.
